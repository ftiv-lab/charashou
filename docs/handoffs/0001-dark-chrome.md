# ハンドオフ 0001 — アプリ外枠を“おしゃれな暗い道具”に（デザイン・トークン適用）

- 作成：Claude / 2026-06-12
- 状態：完了（実装済み・STATUS Done 参照）

## 目的
カード（ピンク）はそのまま、**アプリの外枠（背景・上ツールバー・パネル）を暗くおしゃれに**する。コントラストで“ちゃんとした道具”感を出す。色は**CSSカスタムプロパティ＝デザイン・トークン**に集約し、今後の統一・拡張を楽にする。

## 対象ファイル
- `style.css`（主）
- `index.html`（必要なら左パネルの枠だけ構造追加）
- `app.js` は**触らない**（挙動を変えない）

## やること
1. `:root` に下記トークンを定義。
2. `body` / `.topbar` / `.panel`（新設の左枠）/ `.btn` などを**変数参照**に置き換え（暗い背景＋青アクセント）。
3. レイアウトを **topbar（タイトル＋PNG保存ボタン）＋ 左サイドパネル枠＋ 中央プレビュー** に。左パネルは**枠だけでOK**（中身＝テンプレ/保存カード一覧は後続スライス）。
4. 中央プレビュー周りの余白・角丸・境界を整える。
5. **`#card` 自体の見た目（ピンクの学生証）は変更しない**。

## トークン（そのまま使う・inline）
```css
:root{
  --bg-1:#17191f; --bg-2:#22252e; --bg-hover:#2b303b;
  --panel:#11141c; --border:#383d4a;
  --accent:#3d7bff; --accent-hover:#5c93ff; --warn:#ffc857;
  --text:#f3f5fa; --text-muted:#b0b7c5;
  --radius:6px; --gap:10px;
}
```

## 受け入れ基準
- 暗い外枠＋青アクセント＋左パネル枠＋上ツールバーになっている。
- **カード（#card）の見た目は不変**。
- 既存機能（左入力→即反映・顔写真アップロード・「PNGで保存」）が**すべて動く**。

## スコープ外
- テンプレ切替・カード保存などパネル“中身”の機能（今回は枠だけ）。
- TypeScript化・Vite化・テスト導入（別Phase）。
