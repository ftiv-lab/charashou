# ハンドオフ 0015a — 背景地紋ジェネレーター（編集可能パラメータ）

- 作成：Claude / 2026-06-13
- 状態：完了

## 目的
カード全体の“完成度不足”に一番効く**背景地紋**を、安っぽくならない**編集可能ジェネレーター**にする。テーマ色連動・保存可能・3倍出力で綺麗・権利クリーン。＝「3択プリセットでちゃち」を脱し、**Canva的に自分仕様**へ。

## 位置づけ（確定方針）
- コンセプト＝**創作キャラの身分証メーカー（学生証起点）**。同人/オリキャラの設定資料集に貼る用。ターゲット＝AIは使えるが自作デザインは苦手な層。
- 装飾は **背景/透かし/印＝アプリ生成、校章/紋章＝アップロード＋AIプロンプト補助**（council確定）。本スライスは**その第一弾＝背景地紋**。校章の本格生成は追わない。

## 背景（council：Claude+ChatGPT 一致）
- 背景地紋・透かし・ロゼットは手続き生成でプロっぽく出せる（高ROI・低リスク）。**最初に作るべき1機能＝背景地紋**。
- 本格証券ギョーシェは重い→**軽量版**（反復テキスト/ストライプ/ドット/ロゼットLite）で十分。

## やること
1. **背景 `pattern` を“編集可能ジェネレーター”化**。種類（最低）：
   - `repeatText`（斜め反復テキスト）：text / angle / spacing / opacity / color
   - `stripe`（細線/クロスライン）：angle / spacing / strokeWidth / opacity / color
   - `dots`（ドット/小紋）：spacing / radius / opacity / color
   - `rosetteLite`（円形ライン）：loops / radius / amplitude / strokeWidth / opacity / color
2. **デザインタブに「背景地紋」セクション**：種類セレクタ＋各パラメータ編集（スライダー/数値/色/テキスト）。**プリセットは“出発点”として残す**が、その先を自由に編集できるように。
3. **色は既定でテーマ色連動**（override 可）。
4. **品質ガイド（安っぽくしない）**：細く・控えめ・低不透明度を既定に。1pxベタ塗りの濃い線を避け、上品な密度に。
5. **保存/JSON/Undo-Redo/PNG(3倍)** にパラメータ反映（往復維持）。
6. **学生証の既定背景**を新ジェネレーターで上質に再現（“公的っぽさ”が出る程度に）。

## スコープ外（厳守・後続）
- 丸印ジェネレーターの編集UI（→ 0015b）、校章/印/背景のアップロード＋AIプロンプト補助（→ 0015c）。
- 複数テンプレ（→ 0016）、校章の本格生成、本格証券ギョーシェ、素材同梱。
- 右インスペクタでの背景編集（背景は frame で非クリック＝デザインタブで編集）。

## テスト/検証
- `npm run check` / `check:e2e` 緑。
- 単体：各 pattern → 描画パラメータの純関数／schema往復／パラメータ変更でカードが変わる。
- E2E：種類変更＋パラメータ変更でカードが変わる→保存/リロード復元→PNG反映。
- `npm run shots` 再生成（地紋違いが分かる図）。

## 受け入れ基準
- デザインタブで**背景地紋の種類とパラメータを自由に編集**でき即反映。テーマ色連動。
- 保存/JSON/Undo-Redo/PNG がパラメータ込みで正しく動く。
- 既定の学生証背景が**上質**（ちゃちでない）。
- 既存（4タブ・右インスペクタ・保存/JSON・Undo/Redo・PNG・スクショ）が壊れない。

## 完了後
- `docs/STATUS.md` / `docs/devlog.md` 更新。

## 実装結果
- pattern generatorをrepeatText / stripe / dots / rosetteLiteの判別ユニオンへ拡張し、種類ごとの描画マークを純関数で生成するようにした。
- デザインタブへ背景地紋専用セクションを追加。3プリセットを出発点として、文字・角度・間隔・線幅・半径・ループ数・振幅・濃さ・色を種類別に編集できる。
- 色は既定でテーマの校章アクセントへ追従し、チェック解除で個別色を保存可能。連続編集は既存historyのmergeKeyでまとめる。
- 0014a形式のstripe/dot/wave/rosetteはZod読込後に新パラメータへ補完し、IndexedDB・JSON・Undo/Redo・3倍PNGの往復を維持した。
- Vitest 41件、Playwright E2E 7件を通し、`npm run shots` で反復テキスト地紋のレビュー画像を更新した。
